# 房州ガス デザインプロトタイプ

本番サイトの修正点を事前確認するためのローカルプロトタイプです。

## 使い方

```bash
# 1. このフォルダでコマンドプロンプト/ターミナルを開く
# 2. 以下のコマンドでローカルサーバーを起動
python -m http.server 8888

# 3. ブラウザで開く
# http://localhost:8888/
```

## ファイル一覧

| ファイル | ページ | 主な改善点 |
|---|---|---|
| `index.html` | トップページ | アイコンナビ追加、バナー3列化、お知らせ改善 |
| `gas-application.html` | ガスのご利用 | ステップフロー図解、開栓/閉栓カード |
| `safety.html` | 安全 | 緊急連絡先強調、安全カード、禁止事項アイコン |
| `faq.html` | よくあるご質問 | アコーディオン、カテゴリフィルター |
| `css/override.css` | 共通修正CSS | 横幅1100px、16px、余白統一 |
| `components/step-flow.css` | ステップフロー | STEP 1→2→3コンポーネント |

## 本番への反映方法

1. **CSS変更**: `override.css` の内容を WordPress の「追加CSS」に貼り付け
2. **HTML変更**: 各ページの改善部分を WordPress のカスタムHTMLブロックに配置
3. **画像**: `images/` フォルダの仮画像を本番用画像に差し替え
